RetroControl — DNS Changer for Windows

RetroControl is a lightweight and fast DNS switching tool for Windows. It lets you change your DNS provider instantly without navigating complex system menus.

> [!IMPORTANT]
RetroControl is not a VPN. It only updates your DNS resolver settings using a safe and direct method.

> [!NOTE]
Administrator privileges are required on Windows 7, 8, 8.1, 10, and 11.

Features

One-click DNS switching

Includes popular providers: Cloudflare, Google, Quad9, Yandex, OpenDNS

Instant DNS revert

No background services

Fully portable and open-source

How to Use
Quick Start

Download the ZIP file

Extract it (e.g., C:\RetroControl\)

Right-click RetroControl.exe → Run as administrator

Select your preferred DNS provider

Click Apply DNS

> [!NOTE]
If you move the program folder after applying a DNS profile, use Revert DNS before relocating it.

Restoring Default DNS

Use the Revert DNS button inside the program to restore your original settings.
